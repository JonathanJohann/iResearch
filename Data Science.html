Overview

The field of data science is broad so I'm going to mainly try to focus on a few programming languages and stick to the more exciting analytics. The main languages that I'm going to be focusing on are Python and SQL with some R use. As you're going to see below, there are pro's and con's of each and I try to weave the strengths together to achieve the task at hand.

Note:

  • SQL is not needed to read in these CSV files. I'll be using it to prepare for DB usage down the road in my career.
  • There are a lot of things that you can do in R that you can't in Python. (i.e. prune trees) Until I find a cleaner method that enables R library access from Python, I'll be reading output from text files generated by R scripts. R scrips will be provided in the repo.
